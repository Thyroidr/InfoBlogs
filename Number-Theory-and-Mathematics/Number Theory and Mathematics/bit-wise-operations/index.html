Bitwise Operations

Posted on 2018-10-26 Edited on 2019-04-17 In Number Theory and Mathematics

And Operator: &

Only returns $1$ when both are $1$

Properties

  • $a & b = b & a$
  • $a & a = a$
  • $a & b \le min(a, b)$
  • $a & -a$ returns the right-most $1$ in the binary representation of $a$ (this property is used in binary indexed tree).

The first three properties are obvious, but we would like to show the fourth property. To start with, we should note that,
$$-a = \sim a + 1$$
(Scroll down to see what ~ does)
Notice that adding one to a number sets its first $0$ from right to $1$ and all the $1$ to the right of it to $0$, for example, adding $1$ to $1101$ gives $1110$. Hence, $\sim a + 1$ is different from $a$ from the left until the position of first $1$ of $a$. $a & -a$ would hence give the position of the right-most $1$ in $a$.

Or Operator: |

Only returns $0$ when both are $0$

Xor Operator: ^

Only returns 1 when the bits are different. Xor means exclusive or.

Properties

Mathematically we use the $\oplus$ operator to represent the xor operator.

  • $a \oplus b \oplus b = a$
  • $a \oplus x \ge a - x \forall a \ge x$
  • $a \oplus x = a - x$ iff the bits of $x$ is a subset of the bits of $a$ and $a \ge x$.

(Note: we can denote "the bits of $x$ is a subset of the bits of $a$" by $a & x = x \forall a \ge x$)

Not Operator: ~

The "~" flips the bits of a number, it hence only involves one variable.
