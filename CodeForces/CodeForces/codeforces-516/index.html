1064B: Equations of Mathematical Magic

Problem Link

If $a - (a \oplus x) - x = 0$, then $a - x = a \oplus x$. By the definition of bit-wise xor, $a \oplus x \ge a - x$ and $a \oplus x = a - x$ iff the the bits of $x$ is a subset of the bits of $a$. Hence the answer is $2 ^ {\text{popcount of a}}$.

1064C/1063A: On Those Palindromes

Problem Link

The ends of each palindrome substring must be the same, hence for a letter appearing $x$ times, there can at most $\frac {x (x+1)} {2}$ palindrome substrings ending in it. If we output all of these $x$ letters together, then there are $\frac {x (x+1)} {2}$ palindromes. Hence we can just output the first permutation in dictionary order.

1064D/1063B: Labyrinth

Problem Link

I did not solve this problem during the contest...

I noticed that this problem is a shortest path problem, but I thought it was two dimensional. Instead, optimising for one dimension also optimises for another dimension. (Lesson: should have noticed the data is too large for 2D and there must be ways of optimising to 1D using given information).

The 0-1 BFS mentioned in the official editorial is essentially SPFA. As the edge weights are either 0 or $x$, we can maintain a priority queue in with $O(1)$ time for insertion and deletion. Assume all elements in queue have a distance of either $x$ or $x+k$, then all new added elements have distances of either $x$ or $x+k$.
