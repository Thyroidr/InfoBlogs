FARIO 2014 Connect

Posted on 2019-01-07 Edited on 2019-04-17 In FARIO

Problem Link

Let us consider arrangements such as 1313 where two pairs overlap as edges between node 1 and 3. For any pair of nodes x, we could construct edges to all pairs (with one node in between the x's and the other to the left of the x's). We could do so by iterating from left to right, whenever we encounter the left node of a pair, construct edges to all marked nodes in between it and its right node. We then mark its right node. Since we are always constructing edges to marked nodes in range, we could use segtree to improve the time complexity to $O(n log n)$.
