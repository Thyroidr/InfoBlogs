Combinatorics Tricks

Posted on 2018-10-27 Edited on 2019-04-17 In Number Theory and Mathematics

Assumed Knowledge

  • Combinatorics Function: for $n$ balls, the number of different combinations consisting of $m$ balls can be given by:

    $$ C_n^m = \frac {n!} {m!(n-m)!} $$

  • For a range of size $n$, the number of ways to divide it into $m$ ranges (empty ranges are not allowed) is given by $C_{n-1}^{m-1}$ (in this article $C$ refers to the combinatorics function given above). The situation is equivalent to having $n-1$ gaps and picking $m-1$ of them.

  • Inclusion-exclusion principle: given $n$ sets $A_1, A_2, \ldots, A_n$, we can find $| \bigcup_{i=1}^n |$ using the inclusion-exclusion principle. Consider a Venn diagram with $n$ regions, we are essentially trying to find the total area of the regions. We first add the area of all the regions together. Now, the sub-regions overlapped by at least two regions (call these subregions $s_2$) would be covered at least twice, so we need to reduce them. Now however, the sub-regions overlapped by at least three regions ($s_3$) would have been added $C_3^1 = 3$ times, but also reduced $C_3^2 = 3$ times. $s_3$ has in fact, not been covered, so we need to add $s_3$. Checking $s_4$ now, $s_4$ has been added $C_4^1 + C_4^3 = 4 + 4 + 8$ times, and reduced $C_4^2 = 6$ times, since $8-6=2$, we need to reduce $s_4$. We repeat this process until we reach $s_k = 0$, we know that there are errors with $s_k$, but it is $0$ and the error does not matter (lol). Formally, the inclusion-exclusion principle can be written as,
    $$ | \bigcup_{i=1}^n A_i | = \sum_{0 \neq J \subseteq {1, 2, \ldots, n}} (-1)^{|J|-1} | \bigcap_{j \in J} A_j| $$

On a side note, it is easy to notice that validating the inclusion-exclusion principle can be done by proving the proposition below (I do not know how to),

$f(x) = \sum_{i=1}^{x-1} (-1)^{i+1} C_x^i$ defined over $\mathbb{Z}^{+}$ is equivalent to $f(x) = 2 \times (x+1 \mod 2)$.

Tricks

Let's consider the problem of putting balls into boxes. We can have $2^3=8$ different problems depending on:

  • whether or not the balls are the same (whether or not order matters)
  • consider two boxes, one with $2$ balls and $1$, whether or not putting the box of size $1$ in front matters
  • whether or not empty boxes are allowed

Assume that we have $n$ balls and $m$ boxes.

(I tried to keep the headings in order, but I had to change the order a bit for explanations's sake).

Same balls, box order matters, empty boxes not allowed

In essence, this is the same as the second assumed knowledge, so,

$$ ans = C_{n-1}^{m-1} $$

Same balls, box order matters, empty boxes allowed

We can give each box an additional ball to transform it into the previous trick, since previously empty boxes are not allowed. Hence,

$$ ans = C_{n+m-1}^{m-1} $$

Same balls, box order does not matter, empty boxes allowed

Let $f_{n, m}$ be the number of ways to put $n$ balls into $m$ boxes in this setting.

If there are no balls or only one box, then the answer is clearly $1$.

If $n < m$, then there are guarenteed at least $m-n$ empty boxes and the situation is the same as $f_{n, n}$ since box order does not matter.

If $n \ge m$, then the answer is the number of ways to put $n$ balls in $m$ boxes without empty boxes plus the number of ways to put $n$ balls in $m$ boxes with at least one empty box. The number of ways to put $n$ balls in $m$ boxes can be found be putting one ball in each box in the beginning, and is $f_{n-m, m}$. The number of ways to put $n$ balls in $m$ boxes with at least one empty box can be found by abandonning a box, and is $f_{n, m-1}$. Hence, $f_{n, m} = f_{n-m, m} + f_{n, m-1}$.

Same balls, box order does not matter, empty boxes not allowed

The answer is $f_{n-m, m}$ from the previous section.

Notice that we are essentially looking for the number of partitions of $n$ into $m$ natural numbers. This can be done quicker using the pentagonal number theorem

Diff. balls, box order matters, empty boxes allowed

For each ball, we can place it in any box, hence $ans = m^n$.

Diff. balls, box order does not matter, empty boxes not allowed

See stirling numbers of the second kind.

s2[i][j] = s2[i-1][j] * j + s2[i-1][j-1]

If we are given $j$ non-empty boxes with $i-1$ balls in them in total, then we can put the $i$-th ball in any of them. If we are given $j-1$ non-empty boxes with $i-1$ balls in them in total, then we have to put the $i$-th ball in the $j$-th box.

There exists a formula for the stirling numbers of the second kind, which apparently can be accelerated by FFT (I have not yet learned FFT).

$$ S_2(n, m) = \frac{1}{m!} * \sum_{k=0}^{m} (-1)^k C_k^m * (m-k)^n $$

Diff. balls, box order matters, empty boxes not allowed

The only difference this makes is that order matters, hence,

$$ ans = m! \times S_2(n, m) $$

Diff. balls, box order does not matter, empty boxes allowed

Since we can have empty boxes, we can iterate the number of non-empty boxes and add the answers together.

$$ ans = \sum_{i=1}^m S_2(n, i) $$
