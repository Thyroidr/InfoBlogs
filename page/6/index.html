Q1 & CodeForces 147B: Smile House

Problem Link

Solution

We can rephrase the problem to:

for a given graph of $n$ vertices, find a cycle with the minimum of vertices such that the sum of all edge weights on the cycle is positive.

Let $f_{i, j, k}$ be the maximum distance from $j$ to $k$, passing $i$ vertices. then clearly $f_{i, j, k} = \min{f_{i-1, j, l} + g_{l, k}}$ (memories from Floyd-Warshall). Preprocessing this would have a time complexity of $O(n^4)$, we can hence consider changing $f$ to a sparse table. Let $f_{i, j, k}$ represent the maximum distance from $j$ to $k$, passing $2^i$ vertices, then $f_{i, j, k} = \min {f_{i-1, j, l} + f_{i-1, l, k}}$. This would have a time complexity of $O(n^3 \log n)$.

POI 2015 Pustynia & NOIP 普及 2013 车站分级

Posted on 2018-10-07 Edited on 2019-04-17 In Misc. Problems

Problem Description

The problem "Pustynia" from the Polish POI 2015 is essentially the same problem as the problem "车站分级" from the Chinese NOIP 普及 2013. You can find the original problem in Polish and in Chinese. The Chinese version does have a smaller data size and can be solved with naive methods. Below is the English Translation of the Chinese version.

On a one way rail line, there are $n$ stops labelled from $1$ to $n$. Every stop has a level no lower than $1$. There are multiple train routes using this rail line, all of which satisfy the requirement below:

if a train stops at stop $x$, then it is required to stop at all stops with a level greater than or equall of that of $x$ between its departing and terminal stops (including the departing and terminals).

Freivalds' Algorithm for Matrix Multiplication Verification

Posted on 2018-09-23 Edited on 2019-04-17 In Number Theory and Mathematics

Description

The Freivalds' Algorithm (named after Rūsiņš Mārtiņš Freivalds) verifies whether three given $n \times n$ matrices, $A$, $B$ and $C$, satisfy $A \times B = C$ in $O(kn^2)$, considerably faster than the fastest matrix multiplication algorithm which has a time complexity of $O(n^2.3729)$. It is based on probability, and has a error probability of $2^{-k}$ for a time complexity of $O(kn^2)$.

Euler's Totient Function

Posted on 2018-09-23 Edited on 2019-04-17 In Number Theory and Mathematics

Almighty Euler.

Description

Euler's totient function ($\phi$) maps the positives integers to the number of coprimes below then they each possess. For example, $\phi(6) = 2$. Formally, for $n \in \mathbb{Z}^{+}$,
$$ \phi(n) = n \cdot \prod (1 - \frac{1}{p_i}) \text{ where $p_i$ is a prime factor of $n$}$$

Union Find Set

Posted on 2018-09-21 Edited on 2019-04-17 In Data Structures

Description

Union find set is a tree-based data structure that allows two operations:

  1. query: given $x$, find the root of the tree in which $x$ belongs.
  2. merge: given $x$ and $y$, merge the trees in which they belong (if they do not belong to the same tree).
    (Usually all elements are the root of their own tree in the beginning)

This allows us to answer problems such as:

Outside your hotel window is the town plaza, on which several mafia families have gathered. There are $n$ people in total on the plaza, we know that $m$ pairs of them belong to the same family (different pairs may belong to different families, one individual may be in multiple pairs). Determine the maximum number of mafia families outside.
