FARIO 2012 Happy Feet

Posted on 2018-12-28 Edited on 2019-04-17 In FARIO

Problem Link

Sort all edges by edge weight using count sort. Create a copy of a vertex for each of its edge, and connect edges from the previous copies of u and v to the new copy of both u and v (edges from u to v and from v to u should have edge weights). The resulting graph is a DAG of 2M vertices and 4M edges, we can find the longest path for a DAG in O(M) time using topological sort.

Time complexity: O(M)

Space complexity: O(M)
