Posted on 2018-09-09 Edited on 2019-04-17 In Basic C++

Introduction

Let's say we have the age of 100 students, we can of course declare 100 variables and store this information. This is, obviously, not very effective at all. This is where arrays come into play. An array is a collection of variables that can be accessed by their numerical index. Below is how to declare an array,

Here we declare an integer array named s. it has 100 elements. These elements are accessed by their index. For example, s[0] denotes the first element in this array, and s[2] denotes the third element in this array. The benefit of using arrays is not only that we declare a collection of variables instead of 100 independent variables, but also that we can access them by an index, which is, crucially, a non-negative integer.
