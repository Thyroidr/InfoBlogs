FARIO 2013 Torusia

Posted on 2019-01-03 Edited on 2019-04-17 In FARIO

Problem Link

Let Alison mark $(0, 0), (1, 1), (2, 2) \ldots$.

We know that once Bob found a column with a mark, he could binary search for the column of Alison. To effectively find a column with a mark, we could use the property that the marked columns form a range that is extending to the right. At time $t$, if Bob checked column $x$ with no result, we could check column $x + t + 1$ next since the marked columns (at time $t + 1$) form a range of length $t + 1$.

This will require us at most $91$ checks to find a marked column, as $91 * 92 / 2 = 4186$ (we need to check a bit more since range extends to the right). After we have done the binary search, we could do the same for the rows. Since the marked cells form a diagonal line, when we start checking for the rows, $t$ is large, and that we could check more effectively (about $30$).

FARIO 2012 Master Chef

Posted on 2018-12-28 Edited on 2019-04-17 In FARIO

Problem Link

Let us denote the average tastiness of a range from $l$ to $r$ (inclusive) as $s(l, r)$, and make

$$f(len) = \sum_{l=1}^{n - len + 1} s(l, l + len - 1)$$

We can notice that the sum of average tastiness of all ranges of length $len$ can be found by simply dividing $f(len)$ by $len$. The answer is hence,

$$ans = \frac{\sum_{i=1}^n \frac{f(i)}{i}}{\text{number of ranges}}$$

We know that the number of ranges is $\frac{n (n + 1)}{2}$. We also know that $f(1)$ is the sum of all elements. By noticing that all ranges (except the right-most one) of length $len-1$ are prefixes of ranges of length $len$, we can have,

$$f(i) = f(i - 1) + \sum_{j=i}^n a_j - \sum_{n-(len-1)+1}^n a_j$$

This solves our problem in O(n) time complexity.

DTOJ 4094

Posted on 2018-12-27 Edited on 2019-04-17 In DTOJ

Meow

Phun is a huge playground formed by $n$ pools and $n-1$ paths between pairs of them, such that one can travel from any pool to any other pool. Each paths has a length, determining the number of minutes required to travel between the two pools it connects.

To determine if they are really a match made by God, Meow has invited Neow to Phun. Meow has given $q$ tests to Neow, and each test involves Meow travelling from $u_1$ to $v_1$, and Neow travelling from $u_2$ to $v_2$. Meow will start at the $t_1$-th second, and Neow will start at the $t_2$-th second, they will both travel the shortest paths. Determine, for each test, if Meow and Neow will spend at least one second on the same edge (vertices do not count as part of edges).

Input

In the first line, $n$ and $q$, indicating the number of pools and the number of tests.

In each of the next $n-1$ lines, $u$, $v$, and $w$, representing an edge between pool $u$ and pool $v$ that requires $w$ seconds to travel.

In each of the next $q$ lines, $u_1, v_1, t_1, u_2, v_2, t_2$, representing a test.

Output

For each test, output YES if Meow and Neow will spend at least one second on the same edge, else output NO.

DTOJ 4091

Posted on 2018-12-27 Edited on 2019-04-17 In DTOJ

Dist

Given a connected, weighted, and undirected graph of $n$ vertices. All edges are in exactly one of $k$ special subgraphs, while a vertex is in one or more of these special subgraphs. All edges in a subgraph have the same weight, and each subgraph is complete (for every pair of vertices in the subgraph, there exists an edge in the subgraph connected them). Subgraphs may share vertices.

Find $\sum_{i=1}^n \sum_{j=1}^{i-1} dist(i, j)$ where $dist(x, y)$ denotes the shortest distance from $x$ to $y$.

($1 \le n \le 10^5, 1 \le k \le 18$)

Solution:

We may first find the shortest path from between every pair of subgraphs (let edges have no weight and the weight of each subgraph be the weight of its edges. Iterate over the vertices from $1$ to $n$. When iterating vertex $x$, find the set of subgraphs $x$ is in, call it $S_x$. Sort all subgraphs (including the ones in $S_x$) by there distance to elements of $S_x$. Calculate the contribution of each subgraph to $x$ by increasing order of their distance to $S_x$, and only calculate it for vertices that we have not yet calculated. We can count the number of vertices that in our current subgraph, that has been calculated using the inclusion-exclusion principle and bitwise operators.

DTOJ 4084

Posted on 2018-12-27 Edited on 2019-04-17 In DTOJ

Show

Given an undirected weighted graph, find the number of vertex pairs (a, b) such that all shortest paths from s to t pass through exactly one of a and b ($V \le 50000, E \le 50000$, 2 second, 1GB).

Solution:

Notice that for a valid pair (a, b), the number of shortest paths that path through a or b should be the total number of shortest paths, and that a and b cannot be on the same shortest path. We may count how many shortest paths pass through a vertex, and for each vertex v, which other vertices are on a shortest path with v. By maintaining both using bitset arrays, we may complete bit-wise and, bit-wise xor, and popcount operations in linear time with a 1/32 constant factor, and calculate our answer.
