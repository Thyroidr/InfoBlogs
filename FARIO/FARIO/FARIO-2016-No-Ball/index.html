FARIO 2016 No Ball

Posted on 2019-01-09 Edited on 2019-04-17 In FARIO

Discretise the coordinates first, including the query coordinates.

Iterate lexicographically and use a persistent segtree for 2D range count query. By caching the results for each range that is not fully covered (fully covered ranges have already been cached for) in the initial count query, each "find medium" query can be optimised to $O(q \log n)$.

Space complexity: $O(n \log n)$
Time complexity: $O(n \log n + q \log n)$
