1072C: Cram Time

Problem Link

It is obvious that if $\frac {x(x+1)} {2} > a + b$ then $n+m \neq x$. We can prove that for all $\frac {x(x+1)} {2} \le a + b$, $n+m$ can be $x$. We first need to prove that we can pick some integers (a subset) from $1$ to $x$ such that their sum is exactly $a$. We can do this by going from $x$ to $1$, if $x$ is not smaller than the current integer, then we subtract it from $a$ and add the current integer to the subset, else we add $a$ to the subset (it for sure has not been added to the subset before) and end the loop. This way the sum of the remaining integers is $\frac {x(x+1)} {2} - a \le b$, and we can surely put the remaining integers into $b$.
